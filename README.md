# Industrial Downtime Analysis Dashboard

## Project Overview
This project analyzes machine and factory downtime data to identify operational inefficiencies and device health issues.

## Objectives
- Analyze downtime across factories
- Identify high-risk device types
- Compare performance across companies
- Visualize operational efficiency

## Dataset Includes
- Factory/Company names
- Device types
- Downtime values
- Health status (Healthy/Unhealthy)

## Tools Used
- Excel / Google Sheets
- Tableau

## Key Insights
- Some device types contribute more to downtime
- Certain factories show higher inefficiency
- Clear patterns of unhealthy devices observed

## Outcome
Converted raw operational data into actionable insights using visualization techniques.
